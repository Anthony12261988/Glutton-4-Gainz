This guide shows what accounts to create and what keys you need to share so the Glutton4Gainz app works.

⸻

1. Supabase

What to Do 1. Create a Supabase account and a new project. 2. Go to Settings → API.

What to Share
• Supabase Project URL
• Supabase Anon Key
• Supabase Service Role Key

⸻

2. Google OAuth

What to Do 1. Create a Google Cloud Project. 2. Set up the OAuth Consent Screen (External). 3. Create OAuth Client ID (Web).

What to Share
• Google Client ID
• Google Client Secret

⸻

3. Facebook OAuth

What to Do 1. Go to Meta Developers and create a new App. 2. Add Facebook Login product. 3. Add your redirect URL when asked.

What to Share
• Facebook App ID
• Facebook App Secret

⸻

4. Stripe

What to Do 1. Create a Stripe account. 2. Go to Developers → API Keys. 3. Create or enable the Webhook endpoint later if needed.

What to Share
• Stripe Publishable Key
• Stripe Secret Key
• Stripe Webhook Signing Secret

⸻

5. Resend (Email Service)

What to Do 1. Sign up at Resend. 2. Create an API Key.

What to Share
• Resend API Key

If you verify your domain, share the email address you want to use for sending.

⸻

6. Web Push Notifications (VAPID Keys)

What to Do 1. Generate a VAPID Key Pair using any online tool or ask us to generate.

What to Share
• VAPID Public Key
• VAPID Private Key

⸻

7. Final Checklist (What You Provide to Us)

Please send:
• Supabase URL
• Supabase Anon Key
• Supabase Service Role Key
• Google Client ID + Client Secret
• Facebook App ID + App Secret
• Stripe Publishable Key + Secret Key + Webhook Secret
• Resend API Key
• VAPID Public + Private Keys
• Your live domain (for OAuth and redirects)
